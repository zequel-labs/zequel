# Settings

Zequel provides a settings panel where you can customize the appearance, editor behavior, and data grid display. Open settings by pressing **Cmd+,** (macOS) / **Ctrl+,** (Windows/Linux), or by selecting **Settings** from the application menu.

## Theme

Choose how Zequel appears on your screen:

| Option     | Description                                                        |
| ---------- | ------------------------------------------------------------------ |
| **Light**  | A light color scheme.                                               |
| **Dark**   | A dark color scheme.                                                |
| **System** | Automatically follows your operating system's appearance setting.   |

## Sidebar

- **Sidebar width** -- Adjust the default width of the sidebar panel. You can also resize it by dragging the sidebar edge at any time.

## Editor Settings

These settings control the Monaco code editor used in query tabs:

| Setting          | Description                                                    | Default |
| ---------------- | -------------------------------------------------------------- | ------- |
| **Font size**    | The font size in pixels for the editor text.                   | 14      |
| **Tab size**     | The number of spaces a tab character represents.               | 2       |
| **Word wrap**    | Whether long lines wrap to the next line or scroll horizontally. | Off   |
| **Minimap**      | Show or hide the code minimap on the right side of the editor. | On      |
| **Line numbers** | Show or hide line numbers in the gutter.                       | On      |

## Grid Settings

These settings control the data grid used for displaying query results and table data:

| Setting                  | Description                                                | Default |
| ------------------------ | ---------------------------------------------------------- | ------- |
| **Page size**            | The number of rows loaded per page when browsing table data. | 100   |
| **Alternate row colors** | Apply alternating background colors to rows for easier reading. | Off |

## Applying Changes

Settings are applied immediately as you change them. There is no separate save step. All preferences are persisted locally and restored when you restart Zequel.
